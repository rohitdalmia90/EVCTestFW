# login-regresstion
Verifying the regression suite to test the landing page of all live tenant (including HPU, UVI, CCC, Calhoun, COD, UTA, OKWU, SEARK, UNCFSU, MUW and Coconino) 
To run the test, one just need to download the project locally and to double click on Executable file.
This suit also contains the tenant specific test cases for UVI, HPU, COD, CCC, Calhoun and UTA. This can be run by enabling them.
Prerequisites: 
    Java & Maven must be installed into the system.
Browser version used to run the tests:
    Mozilla Firefox: 57.0(64-bit)
    Google Chrome: Version 76.0.3809.132
